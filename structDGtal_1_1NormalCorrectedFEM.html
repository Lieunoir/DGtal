Detailed Description

template<typename TLinearAlgebraBackend, typename TRealPoint, typename TRealVector>
struct DGtal::NormalCorrectedFEM< TLinearAlgebraBackend, TRealPoint, TRealVector >

Aim: Provides methods for building a stiffness and mass matrix to solve Poisson problems using the Finite Element Method using a corrected normal field.

Description of template class 'NormalCorrectedFEM'

Works with triangular and quadrangular surfaces.

Requires face normals on attached surface mesh.

Template Parameters
TLinearAlgebraBackendlinear algebra backend used (i.e. EigenSparseLinearAlgebraBackend).
TRealPointan arbitrary model of 3D RealPoint.
TRealVectoran arbitrary model of 3D RealVector.

Note
The sign convention for the divergence and the Laplacian operator is opposite to the one of [44] . This is to match the usual mathematical convention that the Laplacian (and the Laplacian-Beltrami) has negative eigenvalues (and is the sum of second derivatives in the cartesian grid). It also follows the formal adjointness of exterior derivative and opposite of divergence as relation \( \langle \mathrm{d} u, v \rangle = - \langle u, \mathrm{div} v \rangle \).
