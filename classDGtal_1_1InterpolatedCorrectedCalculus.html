Detailed Description

template<typename TLinearAlgebraBackend, typename TRealPoint, typename TRealVector>
class DGtal::InterpolatedCorrectedCalculus< TLinearAlgebraBackend, TRealPoint, TRealVector >

Implement differential operators on digital surfaces with a vertex normal field.

Description of template class 'InterpolatedCorrectedCalculus'

Works digital surfaces.

Requires vertex normals on attached surface mesh.

Implements SurfaceDEC methods. Use them to access the corresponding operators.

Template Parameters
TLinearAlgebraBackendlinear algebra backend used (i.e. EigenSparseLinearAlgebraBackend).
TRealPointan arbitrary model of 3D RealPoint.
TRealVectoran arbitrary model of 3D RealVector.
